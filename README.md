# 🚀 TVBox 接口管理中心 (v1.0.31-STABLE)

本项目旨在方便对 TVBox 接口进行统一化管理。您可以轻松地聚合个人收藏的接口、使用本项目推荐的优质接口，并为您生成标准化的订阅链接。

**适配应用**：影视仓、OK 影视、TVBox 原版及各类多仓版 App。

---

## 🌟 核心理念
> 本项目旨在方便对 TVBox 接口的统一化管理，把所有你需要添加的，和本项目推荐的可用接口方便的生成订阅链接，填入影视仓，OK 影视等多仓 App。

---

## 🔑 注册与权限
- **注册邀请码**：`888888` (部署后可在管理控制台修改)
- **管理员权限**：系统默认**第一个注册的用户**即为超级管理员。
- **Webhook 支持**：本项目提供标准 Webhook 接口，可配合 `OpenClaw` 等工具实现全网自动搜索并实时添加可用接口。

---

## 🚀 快速部署 (Docker Compose)

这是最推荐的部署方式，支持数据持久化建议使用下方的配置：

1. **新建目录并创建 `docker-compose.yml`：**
   ```yaml
   version: '3.8'
   services:
     tvbox-manager:
       image: eastar23/tvbox-manager:latest
       container_name: tvbox-manager
       ports:
         - "8089:8089"
       volumes:
         - ./data:/app/data
       environment:
         - REG_CODE=888888  # 默认注册邀请码
         - SECRET_KEY=your_random_secret_key # 建议修改
         - ADMIN_PASSWORD=admin888 # 默认后台登录密码
       restart: unless-stopped
   ```

2. **启动命令：**
   ```bash
   docker-compose up -d
   ```

## ✨ 核心特性
- **🗂️ 接口管理**：支持点播、直播接口分类管理。
- **🖱️ 拖拽排序**：仪表盘支持鼠标/触屏拖拽卡片，实时调整订阅源顺序。
- **🔎 自动检测**：后台自动对接口进行在线检测，状态一目了然。
- **🎁 一键推荐**：内置高品质第三方接口源，支持全量查看、一键添加在线接口。
- **🔗 灵活订阅**：支持单仓 (Legacy) / 多仓 (Standard) 格式导出，支持自动过滤异常源。
- **🛡️ NAS 优化**：针对飞牛 OS、群晖、1Panel 等环境进行了深度适配，低功耗、高响应。

## 🆙 版本更新与维护
- **平滑升级**：数据存储在 `./data` 目录，更新容器镜像不会丢失任何配置。
- **定期维护**：您可以根据需求在管理后台重新生成 Webhook Token 或修改邀请码。

## 🛠️ 关于反馈
如果您在使用过程中发现生成的订阅链接无法下载，请检查您的反向代理配置，或手动设置环境变量 `BASE_URL`。
